蓝牙是一种支持设备短距离通信的无线电技术。能在包括移动电话、PDA、无线耳机、笔记本电脑之间进行无线信息交换。标准是**IEEE802.15**，工作在2.4GHz频带，带宽位1Mb/s。

**要了解引脚和工作模式**

### SM-HC05

AT指令模式（命令响应工作模式）

数据传输模式（自动连接工作模式）

| 引脚  | 说明 |
| ----- | ---- |
| STATE | 配对状态输出：配对成功输出高电平，未配对则输出低电平  |
|RX|模块串口接收脚，接单片机的TXD|
|TX|模块串口发送脚，接单片机的RXD|
|GND|地|
|+5V|电源输入（4.5V\~6V)|
|EN|用于进入AT状态；高电平有效（悬空默认低电平）|

**模块角色：**

- Slave(从角色)——被动连接
- Slace-Loop(回环角色）——被动连接，接收远程蓝牙主设备数据并将数据原样返回给蓝牙主设备

- Master(主角色)——查询周围SPP蓝牙从设备，并主动发起连接，从而建立主、从蓝牙设备间的透明数据传输通道
